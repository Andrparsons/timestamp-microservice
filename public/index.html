Timestamp Microservice

Instructions
  1. Pass in a string as a parameter, it will check to see whether that string is a unix timestamp or a natural language date.
  2. If it does, it returns a JSON object with both the timestamp and the natural language form of the date.
  3. If it does not contain either a date or a unix timestamp, it returns null properties.

Example Usage:

https://tempurl.herokuapp.com/December 15, 2015
https://tempurl.herokuapp.com/1450155600

Example Output:

{"unix": 1450155600,"natural":"December 15, 2015"}
